Keen Caller

Keen caller is a modern react based web application where user can interact with their friends and family.

Technologies used:
- ⚛️ React.js
- 🎨 Tailwind CSS + DaisyUI
- 📊 Recharts (for analytics chart)
- 🔔 React Toastify
- 🌐 React Router

3 key features:

Profile Details:
By clicking on friend profile can see the details about friend.

Call and text:

By clicking on Audio,Vedio and text button a pop up calling toast will show on the top center of the web page,Also the calling history and details stored on the timeline section.

Rechart Dashboard:
In this section total call and text history shows using chart details.
